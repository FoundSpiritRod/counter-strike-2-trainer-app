# Installation Guide

## System Requirements

- Windows 10 / 11 (64-bit)
- .NET Framework 4.8+
- Administrator privileges

## Step-by-Step

1. Download the latest release from the official source.
2. Extract the archive to a folder of your choice.
3. Run the installer or executable as Administrator.
4. Launch the game and then the tool.
5. Press the default hotkey (usually Insert or F5) to open the menu.

## Troubleshooting

**Menu does not appear?**
- Ensure the game is running in Windowed or Borderless Windowed mode.
- Run the tool as Administrator.
- Disable antivirus temporarily (false positive).

**Game crashes?**
- Update to the latest version of the tool.
- Check if the game was updated recently.
- Enable Safe Mode in configuration.

## Uninstall

- Delete the installation folder.
- Remove config files from `%APPDATA%\[ToolName]\`.
