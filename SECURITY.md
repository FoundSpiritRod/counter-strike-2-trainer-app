# Security Policy

## Supported Versions

We actively maintain and update this tool. Please always use the latest release.

## Reporting a Vulnerability

If you discover a security issue:

1. **Do not** open a public issue.
2. Send a private message or email to the maintainer.
3. Describe the issue and how to reproduce it.

We will respond within 48 hours and work on a fix as soon as possible.

## Security Notes

- This tool does not collect or transmit any personal data.
- All modifications are local and client-side only.
- Always download from official sources to avoid malware.
