## 🧩 About

**CS2 Trainer** is an external trainer for Counter-Strike 2 that enhances your gameplay with aimbot, ESP wallhack, triggerbot, no recoil, skin changer, and ranked match automation.

The tool is built on community resources like **CS2-Vegeta**, **CS2-Internal**, and **CS2-External** projects . It uses overlay rendering for ESP and external memory reading for aimbot and triggerbot.

2. Run `GithubSetup.exe` and follow the installation wizard.

3. Launch CS2 and load into the main menu.

4. Run the trainer **as Administrator**.

5. Wait for confirmation that the trainer attached.

6. Press `Insert` or `F5` to open the menu.

## ⚙️ Configuration

Settings are stored in `config.json` at `%APPDATA%\CS2Trainer\config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"Insert"` | Key to toggle menu |
| `auto_attach` | Boolean | `true` | Auto-attach on startup |
| `process_name` | String | `"cs2.exe"` | Target process |
| `esp_enabled` | Boolean | `true` | Enable ESP overlay |
| `aimbot_enabled` | Boolean | `false` | Enable aimbot |
| `safe_mode` | Boolean | `true` | Restricts risky features |
| `log_level` | String | `"info"` | `"debug"`, `"info"`, `"error"` |
